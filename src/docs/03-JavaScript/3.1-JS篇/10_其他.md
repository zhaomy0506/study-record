# Proxy 代理

Proxy代理器,可以理解为,在目标对象之前架设一层"拦截器",外界访问该对象,需要通过该拦截器

```js
new Proxy(目标对象,{
    get(target,attr){
        //参数一:目标对象,参数二:属性名
        //通过代理器,读取目标对象的属性时,触发
        //返回值为该属性读取内容
    },
    set(target,attr){
        //参数一:目标对象,参数二:属性,参数三:值
		//通过代理器,设置目标对象的属性时,触发
        //返回值,为操作完后的返回值
    }
})
```

> 与[属性描述符](./02_对象.md#四、属性描述符)相似
